VLOPs and VLOSEs versions - OpenTermsArchive
============================================

This repository is to be considered as a database only.

It contains the recorded versions of the terms and conditions from Very Large Online Platforms and Very Large Online
Search Engines ([VLOPs and
VLOSEs](https://digital-strategy.ec.europa.eu/en/policies/list-designated-vlops-and-vloses)).

It is recording and archiving versions of terms and conditions as declared in the [`vlop-vlose-declarations
repository`](https://code.europa.eu/dsa/terms-and-conditions-database/vlops-and-vloses/vlop-vlose-declarations).
